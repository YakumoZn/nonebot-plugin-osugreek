# nonebot-plugin-osugreek

在图片上添加osu!mania 4k神秘希腊字母的 NoneBot2 插件

## 介绍

- 在图片中央贴上神秘的4k希腊字母
- 顺便添加色散视觉效果


## 安装

### 方法一：使用 nb-cli（推荐）
留空

### 方法二：从 GitHub 安装
pip install git+https://github.com/YakumoZn/nonebot-plugin-osugreek.git安装git https://github.com/YakumoZn
onebot-plugin-osugreek.git


## 使用

### 基础命令
/osugreek <希腊字母名称>   /osugreek <希腊字母名称>
或
/希腊字母 <希腊字母名称>

### 使用方式
回复图片消息并输入：/osugreek <希腊字母名称>
